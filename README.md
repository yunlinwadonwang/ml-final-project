# ml-final-project

## Requirements

To install requirements:

```setup
pip install -r requirements.txt
```

## Training

To train the model, run this command:

```train
python 109550038_Final_train.py
```

## Evaluation

To evaluate my model, run:

```eval
python 109550038_Final_inference.py
```
